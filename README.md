# TuneTwist Chrome Extension

## Installation

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/ArbazkhanPathan/TuneTwist.git
Open Google Chrome and go to chrome://extensions/.

Enable "Developer mode" in the top right corner.

Click on "Load unpacked" and select the directory where you cloned the repository.

The TuneTwist extension should now be installed and visible in your Chrome extensions.

## Usage
Once the extension is installed, follow these steps to use TuneTwist:

Open a website or article that you want to transform into song names and lyrics.

Click on the TuneTwist extension icon in the Chrome toolbar.

Wait for 7-8 seconds as the extension processes and transforms the content of article.

The changes will be reflected in red color boxes within the article, showing the modified headings and lyrics.

## Notes
The extension may take 7-8 seconds to fully process and reflect changes on the website.

Modified content will be highlighted in red boxes to distinguish it from the original text.
